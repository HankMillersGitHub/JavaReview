### 4. 流程控制

#### 4.1 if-else

```java
// if的第一种格式
if(条件){
    执行语句
}
// if的第二种格式
if(条件){
    执行语句1
}else{
    执行语句2
}
// if的第三种格式
if(条件1){
    执行语句1
}else if(条件2){
    执行语句2
}else{
    执行语句3
}
```

#### 4.2 switch

```java
switch (变量) {
    case 值1 : 执行语句1;break;
    case 值2 : 执行语句2;break;
    case 值3 : 执行语句3;break;
    default: System.out.println('0');
}
```

#### 4.3 for

```java
for(初始化;条件;步进){
    循环体
}
```

#### 4.4 while(可能一次都不执行)

```java
初始化
while(条件控制){
    循环体
}
```

#### 4.5do-while(必执行一次)

```java
初始化
do{
    循环体
}while(条件)
```
